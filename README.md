# Pyatform

Pyatform Frontend is a web client for Pyatform - a coding challenge platform.
It allows users to write code, submit solutions, view test results, track progress, and manage their profile.

The frontend communicates with the [.NET backend API](https://github.com/arekminajj/pyatform-backend)
## Getting Started

First, run the development server:

```bash
npm run dev
```

The app will be available at:
```
http://localhost:3000
```
